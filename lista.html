Unordered list can have a type attribute which can be applied to the UL or LI tag. The tree values allowed are:

  • type = "square"
  • type = "disc"
  • type = "circle"

Ordered lists can have a type attribute too. Used along with a start attribute they can show

  1. Lower case letters
  1. Numbers
